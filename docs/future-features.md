## Planned Feature: Pack Opening Analytics

### Motivation

Collectors often obtain cards by opening booster packs instead of purchasing singles.

Future versions of the application will support recording pack opening history to analyze:

- Pack opening cost
- Expected Value (EV)
- Pull distribution
- ROI
- Set profitability
